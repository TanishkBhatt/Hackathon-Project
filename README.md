# SMART Study-Life Planner

---

## Introduction
-`SMART Study-Life Planner` is a KVS Hackathon Project created by Team `D1388JR1` of PM SHRI Kendriya Vidyalaya Pithoragarh. It’s a simple interactive planner that helps users organize study routines, tasks, and priorities effectively within a study-life balance framework.

---

## Live Preview
View the live deployed version here:  
https://tanishkbhatt.github.io/Hackathon-Project/ 

---

## Features
- Interactive study planner interface
- Task creation and scheduling support
- Prioritization of study activities
- User-friendly design for quick planning and navigation
- Works responsively on modern browsers

---

## Tech Stack Used
This project is built with:
- `HTML5` — Structure of the web pages
- `CSS3` — Styling and layout
- `JavaScript` — Interactive functionality
